# Guidelines for burning for Burners

This repository contains working Markdown guidance for planning and running art burns at burner events.

The current document set focuses on Burning Art at Borderland. It turns the main guideline into AI-friendly Markdown and separates operational checklists into maintained files so burn plans can link to them instead of duplicating checklist content.

## Intended workflow

1. Read the guideline to understand requirements.
2. Copy the burn plan example template for a specific artwork.
3. Assign named people to the required roles.
4. Link each role to the maintained checklists relevant to that role.
5. Use the master operational checklist and go/no-go checklist to coordinate readiness and execution.

Generated source documents, draft burn plans and artwork-specific plans should be kept separate from the maintained guideline and checklist files unless they are intentionally ready to publish.
